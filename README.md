# API-сервис для инференса модели машинного обучения

Модель машинного обучения в формате ONNX (`diabetes_model.onnx`). Вывод вероятности наличия диабета у пациента по признакам.

Признаки пациента:
- `Pregnancies` (количество беременностей),
- `Glucose` (уровень глюкозы),
- `BMI` (индекс массы тела),
- `Age` (возраст).

Вывод:
- если вероятность > 0.5 → предсказание 1 (есть диабет)
- иначе 0 (нет диабета).


## Запуск
- Локальный (poetry)
```poetry run uvicorn src.entrypoints.fastapi_app:app --reload```
- Docker
```docker compose up --build```

Доступ к запросу вероятности диабета у пациента (/predict) есть только у авторизованных пользователей
